## ⚠️ 二、必须遵守的注意事项

### 2.1 安全红线

| 事项 | ❌ 错误 | ✅ 正确 |
|------|---------|---------|
| 密码 | `password = "123456"` | `BCryptPasswordEncoder.encode(password)` |
| 权限 | 无权限注解 | `@PreAuthorize("hasRole('HR_ADMIN')")` |
| Hive SQL | 字符串拼接 `"SELECT * FROM " + tableName` | 使用占位符 `WHERE period = ?` |

### 2.2 数据质量

| 事项 | 正确处理 | 位置 |
|------|----------|------|
| 空值 | `row.put("value", 0.0f)` | DataCleanService |
| 异常值 | `if (value > 100.0f) value = 100.0f` | DataCleanService |
| Hive 分区 | `WHERE period = '2026'`，禁止全表扫描 | HiveAnalysisEngine |
